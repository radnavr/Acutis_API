ACUTIS API

Věděli jste, že patronem programátorů a internetu je blahoslavený Carlo Acutis? Pokud ne, tak jste na správném místě, protože Acutis API je ultimátní branou do Acutisverza.
Acutis API nabízí routes, které poskytují JSON array jeho skutků či JSON array odkazů na obrázky, související především s jeho posmrtným životem. Kromě těchto arrays ale můžete získat i jeden náhodný skutek či jeden náhodný obrázek.

// GET all images

http://localhost:5000/img/all

// GET one random image

http://localhost:5000/img/one

// GET all deeds

http://localhost:5000/deeds/all

// GET one random deed

http://localhost:5000/deeds/one
